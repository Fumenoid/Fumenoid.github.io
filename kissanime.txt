      <h3 style="margin-top:55px;margin-left:25px;" class="w3-xxxlarge w3-text-teal"><b>How it works ?</b></h5>
	  <hr style="width:45%;border:4px solid #009688;margin-left:20px;" class="w3-round w3-text-teal">
      <h5 style="margin-left:40px;">Before you just jump into this, I hope you have a basic understanding of html, css and Javascipt, Also you should now basics behind browser extensions and their anatomy.</h5>
      <h3 style="margin-top:15px;margin-left:40px;" class="w3-xxlarge w3-text-teal"><b>Javascipt</b></h5>
	  <hr style="width:25%;border:4px solid #009688;margin-left:35px;" class="w3-round w3-text-teal">
	  <h5 style="margin-left:55px;">The Concept behind this extension, or in general behind any algorithm that works on removing adverts is to remove the tags with id containing "ad" or "ads", and hyperlink tags that do nothing from the DOM structure. Easy enough right, nah, it's a hard task to design an algorithm to do that because you need to take care that you do not remove the real content from the website.</h5>
      <h5 style="margin-left:55px;">Okay so this extension works on content of a webpage, so we need to add.</h5>
      <h5 class="w3-light-grey" style="margin-left:55px;">"content_scripts": [</br>
    {</br>
      "run_at": "document_idle",</br>
      "matches": ["*://*.kissanime.ru/*","*://*.kissmanga.com/*","*://*.novelplanet.com/*"],</br>
      "js": ["advertHider.js"]</br>
    }</br>
  ]</br></h5>
  <h5 style="margin-left:55px;">In Manifest.json file "content_scripts" because it will only work on the content of certain webpage or websites only, "matches" makes this Javascipt to work on only those pages whose link is mentioned, and "js" has the file name of the Javascipt code.</br></h5>
  <h5 style="margin-left:55px;">Basically when the url in url box will match the expression of the website, the javascript mentioned in "js" will work on the content that the specified urls return.</h5>  
  <h5 style="margin-left:55px;"></br>The basic Javascipt code.</h5>
  <h5 class="w3-light-grey" style="margin-left:55px;"> const [...sidelayelem] = document.querySelectorAll('div[id*="ad"]')</br>
    sidelayelem.forEach(elem => {</br>
      elem.parentNode.remove()</br>
    })</br>
    </h5>
    <h5 style="margin-left:55px;">This code will delete all the div from DOM, whose id contains "ad", where ad means adverts as generally has term ad in their id's.</br> Note:The code in the Extension is quite complex compared to this, it's just a general representation of what is happening, you can read the complete code at github.</h5>
