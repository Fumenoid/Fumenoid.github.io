      <h3 style="margin-top:55px;margin-left:25px;" class="w3-xxxlarge w3-text-teal"><b>How it works ?</b></h5>
	  <hr style="width:45%;border:4px solid #009688;margin-left:20px;" class="w3-round w3-text-teal">
      <h5 style="margin-left:40px;">Before you just jump into this, I hope you have a basic understanding of html, css and Javascipt, Also you should now basics behind browser extensions and their anatomy.</h5>

    <h3 style="margin-top:15px;margin-left:40px;" class="w3-xxlarge w3-text-teal"><b>Manifest.json file.</b></h5>
	<hr style="width:25%;border:4px solid #009688;margin-left:35px;" class="w3-round w3-text-teal">
    <h5 style="margin-left:55px;">The Manifest.json file contains the basic information about the Browser Extension, such as Extension Name, Version, Manifest version, Description of the browser extension and Icon details</h5>
    <h5 class="w3-light-grey" style="margin-left:55px;">  "manifest_version": 2,</br>
  "name": "What's My IP Address",</br>
  "version": "1.1.0",</br>
  "description": "A browser extension that displays your IP address and your location based on your IP address, you can use it to check your vpn(proxy) or your public IP Address just just a click :D",</br>
  "icons":{</br>
    "48":"icon/micon.png"</br>
  }</br>
  </h5>
    <h5 style="margin-left:55px;"></br>But this browser extension also requires a UI so that the user can request the api to verify his IP Address.</h5>
    <h5 class="w3-light-grey" style="margin-left:55px;">   "browser_action": {</br>
    "default_title": "FetchmyIP",</br>
    "default_popup": "popup/Smokes.html",</br>
    "default_icon": {</br>
      "32": "icon/button/30pxbicon.png"</br>
    }</br>
  },</br>
  </h5>
  <h5 style="margin-left:55px;">This will tell the browser to make a button from which the UI interface can be accessed.</h5>
  <h5 style="margin-left:55px;">The file "smokes.html" will contain the gui, and the Javascipt to do the web request, but before this we need to make add the permissions in the manifest.</h5>
      <h5 class="w3-light-grey" style="margin-left:55px;">   "permissions": [</br>
    "*://api.ipgeolocation.io/*", </br>
    "webRequest"</br>
  ],</br>
  </h5>

    <h3 style="margin-top:15px;margin-left:40px;" class="w3-xxlarge w3-text-teal"><b>The Popup.</b></h5>
	<hr style="width:25%;border:4px solid #009688;margin-left:35px;" class="w3-round w3-text-teal">
    <h5 style="margin-left:55px;">The Popup contains html, css and Javascipt behind the browser extension. The html and css is preety normal of this extension, you know if you used it, i generated the image from Wordart.com, to avoid any copyright issues : )</h5>
    <h5 style="margin-left:55px;">The Javascipt code does an api request to the web api of ipgeolocation.io and as it is a CORS, Cross origin resource sharing i asked for permission earlier in the manifest.json file.</br>The Returned data should got stored in a table after converting it into text. Also added try and catch in case the web request fails.</h5>
